# Angular Slip List

一個清單列表刪除具有視覺效果的組件。

## Design

靈感在自 [UI Movement](https://uimovement.com) 的 [Delete Message](https://uimovement.com/ui/6311/delete-message/)。

## Installation

```bash
git clone https://github.com/explooosion/angular-slip-list.git
```

```bash
cd angular-sli-list && yarn
```

```bash
yarn start
```
